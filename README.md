# magika-dorue

あなたは多くのあくまを配下に持つ魔王です。地下にあくまの根城を作るため、地面を掘って、モンスターを生み出し、上から攻めてくる勇者を叩き潰しましょう！勇者を倒して、より多くのゴールドを得ることで、より強い自分だけの王国を作りましょう！

---

You are the Demon Lord, commanding a vast army of monsters. Dig deep into the earth to build your Evil Chamber, unleash monsters from the rock, and crush the heroes who dare to invade from above! Defeat heroes, earn gold, and forge your own unstoppable underground kingdom!

---

## 遊び方 / How to Play

- 地面をクリックして掘り、モンスターを解放する / Click tiles to dig and release monsters
- あくま部屋を設置して勇者の侵攻を開始する / Place the Evil Chamber to start the hero invasion
- モンスターをレベルアップ・強化して防衛力を高める / Level up and upgrade monsters to strengthen your defense
- あくま部屋を深く移動させてダンジョンを広げる / Move the Evil Chamber deeper to expand the dungeon infinitely
- 勇者を倒してゴールドを稼ぎ、次のゲームに引き継ぐ / Defeat heroes to earn gold carried over to the next run

## モンスター / Monsters

- 🪨 **石スライム / Stone Slime** — 耐久力が高くタンク向き / Tough and slow, great for tanking
- 🌿 **苔ゴブリン / Moss Goblin** — 素早く索敵範囲が広い / Fast with a wide search range
- 🟤 **銅ゴーレム / Copper Golem** — 攻撃力と耐久力に優れる / Powerful and durable frontline fighter

## 特徴 / Features

- 無限に広がる手続き生成ダンジョン / Infinite procedurally generated dungeon
- モンスター繁殖システム / Monster breeding system
- ゴールド・モンスターレベルの引き継ぎ / Gold and monster level carry-over between runs
- 勇者タイプ・装備システム / Hero types and equipment system
- 早送りモード / Fast-forward mode
